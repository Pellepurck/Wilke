Bærekraft

I dagens samfunn er det økende krav og forventninger om å ta ansvar for bærekraftige handlinger, og dette gjelder for alle – uansett størrelse eller omfang av virksomheten din.

Lovgivningen endres kontinuerlig, og stadig nye krav blir stilt til bærekraftig praksis. Din tilpasningsevne til disse endringene kan påvirke tilgangen til kapital, forsikring og partnerskap.

Å ikke ta bærekraft på alvor kan resultere i økonomiske tap, omdømmeskader og negative effekter av eksterne hendelser som klimaendringer. I tillegg kan mangel på bærekraftspraksis svekke tillit og tilgang til markeder og arbeidskraft.


Vi kan alle bidra – uansett størrelse


Mitt mål å hjelpe små og mellomstore bedrifter med å forstå hvordan de kan bidra til bærekraft og hvordan det kan implementeres på en fungerende måte i hver unik bedrift. Bærekraft handler om å finne riktig tilnærming for deg og din virksomhet, det handler ikke om hvor stor eller liten din virksomhet er, men om viljen til å delta og gjøre en forskjell.

Uansett hvor stor eller liten din virksomhet er, kan du bidra på din egen måte. La oss ha en samtale om hvordan vi sammen kan finne den rette veien for deg.
